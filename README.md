# 社区流浪动物救助管理系统 java698

## 项目概述

社区流浪动物救助管理系统是基于Spring Boot开发的B/S架构系统，旨在为社区提供一个便捷、高效的流浪动物救助管理平台。

## 技术栈

- **前端技术**：Bootstrap、HTML、CSS、JavaScript、jQuery、Thymeleaf
- **后端技术**：Spring Boot、MyBatis
- **数据库**：MySQL 5.5/5.7/8.0
- **开发环境**：IntelliJ IDEA
- **运行环境**：Windows 7 或 Windows 10、JDK 1.8

## 系统角色

系统分为以下两个角色：

- **管理员**
- **普通用户**

## 功能模块

### 管理员

1. **登录**
2. **个人信息管理**
3. **用户管理**
4. **动物类别管理**
5. **帖子管理**
6. **动物管理**
7. **公告管理**
8. **动物领养管理**
9. **动物捐赠管理**
10. **评论管理**
11. **退出登录**

### 普通用户

1. **登录注册**
2. **浏览动物信息**
3. **查看动物详情**
4. **爱心捐赠**
5. **查看公告**
6. **发布帖子**
7. **查看帖子详情**
8. **评论帖子**
9. **个人信息管理**
10. **申请领养宠物**
11. **历史记录查询**

## 运行环境

- **操作系统**：Windows 7 或 Windows 10
- **Java开发工具包**：JDK 1.8
- **数据库**：MySQL 5.5/5.7/8.0

## 部署步骤

- 确保环境满足运行要求
- 使用Maven构建项目
- 导入数据库文件
